# ft_menuExemple

What things you need to install this script them

 - [ft_menuBuilder](https://github.com/FivemTools/ft_menuBuilder)

 Put the folder where you want it and add it to the citmp-server.yml file

 Example:

 ```
 - ft_menuExemple
 ```
